READ_ME__________________________________________________________

Note: This is a tool to practice Typing 

Precaution:
CapsLock must be Off before clicking on the start button

STEPS TO START PRACTISING:

Step1: Run the project.html file in a Javascript Enabled Browser(e.g. Google Chrome)
Step2: The webpage opens and you can click on the 'START' button
Step3: Now, you may start practicing typing.For each correct letter you press, you see the letter converted to green color.
		If, it has been typed wrong, then , you need to backspace and correct it.
Step4: In case, if you want to change the text being displayed in the webpage,you can go to the foo.txt file,
		present in the folder and modify its contents.
